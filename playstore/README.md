Play Store assets and metadata placeholders

This folder contains draft metadata and lightweight placeholder graphics to help prepare a Play Console submission.

Files
- `title.txt` — app title
- `short_description.txt` — short description for the Play Store listing
- `full_description.txt` — long description
- `release_notes.txt` — release notes / what's new
- `feature_graphic.svg` — 1024x500 feature graphic placeholder (SVG)
- `icon.svg` — 512x512 adaptive icon placeholder (SVG)
- `screenshots/` — simple placeholder screenshots (SVG)

Next steps
- Replace placeholder SVGs with final PNG/JPG assets at the sizes required by the Play Console.
- Edit the textual metadata files to match marketing copy.
- Use the Play Console or `bundletool` / `gcloud` to upload artifacts when ready.
