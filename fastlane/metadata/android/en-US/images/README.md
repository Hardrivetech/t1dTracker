Fastlane images placeholder

Place Play Store screenshot images in PNG format in this folder before running `fastlane supply`.

Recommended filenames (example):
- phone_screenshot_1.png
- phone_screenshot_2.png
- phone_screenshot_3.png
- tablet_screenshot_1.png

Current repo contains SVG placeholders in `playstore/screenshots/` which you should replace with high-quality PNGs.
